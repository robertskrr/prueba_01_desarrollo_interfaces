Gestor de Visitas y Temperaturas

Esta aplicación de escritorio, desarrollada en Java con Swing (javax.swing.JFrame y JDialog), permite a los usuarios registrar, visualizar y gestionar información sobre sus viajes o visitas.

El programa administra una lista de objetos Visita, cada uno de los cuales contiene la Ciudad, la Fecha prevista, la Estación del año, y las Temperaturas (mínima, máxima y promedio) registradas.

La interfaz principal muestra un listado tabular (JTable) de todas las visitas, permitiendo al usuario:

  • Visualizar los datos de las visitas.
  • Ordenar las visitas, por defecto por la temperatura promedio de forma descendente.
  • Cambiar el aspecto visual de la aplicación (Look and Feel).
  • Acceder a formularios de registro para nuevas visitas o para añadir temperaturas a la última visita registrada.
  • Eliminar registros de visitas por nombre de ciudad.

La aplicación incluye validaciones en los formularios de registro (usando la librería NetBeans Validation API) para asegurar la integridad de los datos.
